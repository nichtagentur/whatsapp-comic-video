# WhatsApp clip → Comic video

WhatsApp clip stylized as comic-book animation via OpenRouter video API.

Pipeline:
1. ffmpeg first-frame extract
2. Gemini 2.5 Flash Image (nano-banana) — comic-book restyle
3. ByteDance Seedance 2.0 Fast img2vid (5s, 720x1280)
